# The Party Hub 🎉

The Party Hub is a modern multi-app entertainment and lifestyle platform built with Next.js, React, and Tailwind CSS.  
The application combines multiple interactive experiences into one platform, including cocktail discovery, movie browsing, and restaurant menu exploration.

Designed with a clean UI and responsive layout, The Party Hub delivers a fast and smooth experience across desktop, tablet, and mobile devices.

---

# Live Features 🚀

## 🍸 Cocktail App
- Search cocktails by name
- View cocktail details and ingredients
- Dynamic cocktail pages
- Responsive cocktail gallery UI
- Loading states for better UX

## 🎬 Movies App
- Browse movie collections
- Explore movie cards and details
- Modern responsive movie layouts
- Smooth navigation experience

## 🍔 Menu App
- Interactive restaurant-style menu
- Category-based filtering
- Responsive food cards
- Clean and modern design
